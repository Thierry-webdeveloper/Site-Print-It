# Print it! — Carrousel JavaScript

Projet 5 de la formation **Intégrateur Web** — OpenClassrooms.

## 🎯 Objectif du projet

Dynamiser le site statique d'une imprimerie familiale (_Print it!_) en intégrant un **carrousel interactif** en JavaScript dans la bannière d'accueil.

## ✨ Fonctionnalités implémentées

- Navigation entre les slides via des flèches gauche / droite
- Bullet points synchronisés avec la slide active
- Défilement infini en boucle
- Mise à jour dynamique de l'image et du texte à chaque changement de slide

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript
